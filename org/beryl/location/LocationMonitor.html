LocationListener, ContextClonable
public class LocationMonitor
extends Object
implements LocationListener, ContextClonable

Monitors multiple location providers at the same time and routes the readings to all listeners. This type is ContextClonable but listeners that are not ContextClonable will need to be reattached.

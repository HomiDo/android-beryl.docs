Interface IUrlShortener

  • All Known Implementing Classes:
    AbstractUrlShortener, TinyURLUrlShortener, UrlyUrlShortener, VgdUrlShortener

    public interface IUrlShortener
    Interface for URL Shortening providers. URL Shortening services take very long URLs and converts them to an auto-redirecting short URL appropriate for posting data on sites like twitter. Requires: android.permission.INTERNET

Package org.beryl.intents.android Description

Collection of Intent generating methods that typically can be handled by the stock android system. Some intents request for a result. In order to handle the result you'll need to override onActivityResult within the launching Activity. There's typically a convienience method to consume the returning intent for use. These are named onActivityResult_[MethodName].

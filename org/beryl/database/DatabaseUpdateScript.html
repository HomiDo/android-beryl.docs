Class DatabaseUpdateScript

  • java.lang.Object
    • org.beryl.database.DatabaseUpdateScript

    public abstract class DatabaseUpdateScript
extends Object
implements IDatabaseUpdateScript
    Base class for database update scripts. Assumes that the schema patch is in the assets/ directory as an .sql file. This base class also provides convenience methods for performing database updates.
